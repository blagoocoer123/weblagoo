# Version 5.3

## Изменения:
- **ИСПРАВЛЕНО:** Player Modal больше не открывается при скролле по пустому месту
- Добавлен `pointer-events: none` для неактивного Player Modal
- Player Modal теперь блокирует клики только когда активен
- Улучшена обработка touch-событий для модального окна

## Предыдущие изменения (v5.2):
- Оптимизация для мобильных устройств
- Исправлена проблема с случайным открытием видео при скролле
- Добавлена защита от случайных кликов во время прокрутки
- Улучшена адаптивность Dynamic Island и Player Modal для телефонов
- Отключен 3D эффект наклона карточек на мобильных устройствах
- Добавлена правильная обработка touch-событий
- Видео теперь автоматически останавливается при закрытии правой панели
- Улучшена видимость и расположение элементов на маленьких экранах
- Добавлен `pointer-events` контроль для предотвращения конфликтов при скролле
- Оптимизирован z-index для правильного наложения элементов

## Технические улучшения:
- Touch-события с проверкой движения (менее 10px = клик, иначе скролл)
- Добавлен `touch-action: pan-y` для body на мобильных
- Видео элементы получили `pointer-events: none` для предотвращения блокировки скролла
- Улучшена структура z-index для мобильной версии
- Player Modal не перехватывает события когда неактивен
