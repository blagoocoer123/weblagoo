# Version 5.8

## Исправления:
- **ИСПРАВЛЕНО:** Регулятор громкости теперь работает на мобильных устройствах
- Добавлена обработка событий `change` и `touchmove` для volume bar
- Увеличен размер ползунка громкости на мобильных (20px вместо 14px)
- Добавлен `touch-action: none` для volume bar для лучшей работы на touch-устройствах
- Увеличена высота полосы громкости на мобильных (8px вместо 6px)
- Улучшена touch-поддержка для всех слайдеров

## Предыдущие изменения (v5.7):
- ИСПРАВЛЕНО: Кнопки "Инфо" и "О себе" больше не налазят на заголовок на мобильных
- Кнопки переключения табов перенесены вниз карточки

## Предыдущие изменения (v5.4):
- ПОЛНОСТЬЮ ИСПРАВЛЕНО: Правая панель с видео теперь вообще не существует в DOM, пока не нажата стрелочка
- Видео не загружается и не появляется до нажатия на кнопку
- Стрелочка поднята выше на мобильных устройствах
